> * git init :初始化一个版本仓库

> * git clone  address       克隆某一个文件

> * git remote -v       查看远程仓库


--------------提交修改

> * git add .   添加当前修改的文件到暂存区

> * git commit –m &quot;你的注释&quot;         提交你的修改

> * git remote add origin yourgitaddress(你的git地址)     添加到远程版本库

> * git push origin master     推送到远程服务器

> * git status      查看文件状态
